Main idea

Simple Angular accordion component and a simple demostrator app to use in Angular/Ionic apps; This is a simple demonstrator project - to be consumed/used as is (links to the original repository are must).

Any suggestions for improvement are welcome. Please file a new issue if you have any problems using this package or find ways in which it could be improved. To ask for features / report on a bug follow the guide in : Issue Report
